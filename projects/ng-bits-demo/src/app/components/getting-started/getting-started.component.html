Getting Started

Installation

Install using one of the following:

  
    $ ng add ng-bits
    $ npm i ng-bits
    $ yarn add ng-bits
Usage

Import the module into your app.module.ts file

   
    {{import}}

    {{ngModule}}
    {{imports}}
    {{ngModuleEnd}}
    {{export}}
