# NgBits

## Installation

Install using one of the following:

```bash
$ ng add ng-bits
$ npm i ng-bits
$ yarn add ng-bits
```

## Usage
Import the module into your `app.module.ts` file

```ts
import { NgBitsModule } from 'ng-bits'

@NgModule({
  imports: [ NgBitsModule ]
})
export class AppModule {
}
```
